# Banking Dashboard Project

## Project Purpose
A simple banking dashboard UI for learning web development.

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Chart.js (via CDN) for charts

## File Structure
- index.html — main page structure
- style.css  — all styling
- app.js     — charts and data logic

## Design Theme
Dark navy and gold. Professional banking aesthetic.

## Developer Notes
- No frameworks, no npm, no build tools
- Must open directly in browser (no server needed)
